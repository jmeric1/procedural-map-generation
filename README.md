# Procedural Map Explorer
**Author:** John Meric

## Overview
This is a 2D game engine built in Python using Pygame. It uses **Cellular Automata** to generate organic, cave-like structures.

## Key Features
* **Procedural Generation:** Uses a smoothing algorithm to create unique maps every time.
* **Seed System:** Players can input a custom string to share specific map layouts.
* **Game State Logic:** Includes a functional Start Menu, Game HUD, and win conditions.
* **Custom UI:** Hand-written event loops for text input and continuous movement.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the game: `python3 generator.py`
